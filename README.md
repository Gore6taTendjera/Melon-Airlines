# Melon Airlines

Melon Airlines is a Web Application for ticket booking with advanced seat allocation algorithm. The system prioritizes aisle seats first, followed by middle seats, and then window seats, filling from the front rows to the back to optimize weight distribution. Passengers also have the option to book a specific seat directly, which increases the price of the ticket for each seat class: Economy, Business, and First. Supports A320 & A380 (2 floors) seat configurations.


## Tech Stack

- **Frontend:** HTML CSS JavaScript | ASP.NET Core Razor Pages
- **Backend:** ASP.NET Core
- **Database:** Microsoft SQL Server (MSSQL)
- **Testing:** MSTest
